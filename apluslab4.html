PowerShell Networking Commands.



    Basic Networking PowerShell cmdlets


  1. Get the IP Configuration (ipconfig with PowerShell):  Get-NetIPConfiguration


  2. List all Network Adapters:  Get-NetAdapter | format-list


  3. Get a specific network adapter by name :  Get-NetAdapter -Name *Ethernet* | format-list


  4. Get more information VLAN ID, Speed, Connection status :  Get-NetAdapter | ft Name, Status, Linkspeed, VlanID


  5. Get driver information:  Get-NetAdapter | ft Name, DriverName, DriverVersion, DriverInformation, DriverFileName | format-list


  6. Get adapter hardware information. This can be really usefull when you need to know the PCI slot of the NIC.:  Get-NetAdapterHardwareInfo


  7. Disable and Enable a Network Adapter (assuming that the name is Wi-Fi, yours may be different):  Disable-NetAdapter -Name "Wi-Fi" Enable-NetAdapter -Name "Wi-Fi"


  8. Rename a Network Adapter (requires elevation!):  Rename-NetAdapter -Name "Wi-Fi" -NewName "Wireless"




    IP Configuration using PowerShell


  1. Get IP and DNS address information :  Get-NetAdapter -Name "Ethernet" | Get-NetIPAddress


  2. Get IP address only :  (Get-NetAdapter -Name "Ethernet" | Get-NetIPAddress).IPv4Address


  3. Get DNS Server Address information :  Get-NetAdapter -Name "Ethernet" | Get-DnsClientServerAddress


  4. Set interface to DHCP (requires elevation) :  Set-NetIPInterface -InterfaceAlias "Wireless" -Dhcp Enabled


  5. List DNS Cache: :  Get-DnsClientCache


  6. Clear DNS Cache :  Clear-DnsClientCache


  7. List DNS Cache (notice the difference) :  Get-DnsClientCache




    Ping with PowerShell


  1. Simple ping :  Test-Connection intelligence.gov


  2. Another way to ping :  Test-NetConnection -ComputerName intelligence.gov


  3. Yet another way :  Test-NetConnection -ComputerName intelligence.gov -InformationLevel Detailed
