Working with common Linux commands.


We'll use an online JavaScript emulator created by Fabrice Bellard (google him) to run a lot of Linux bash commands. It's not the whole 9 yards, but it's a good alternative to a Linux virtual machine to get started with basic commands. In the future we'll create a Linux VM (an easy task to accomplish). In the meantime visit this website, wait half a minute and start typing the commands shown below. Try to make sense of what you're writing. Remember that LMGTFY is out there to help you.
