Our first Python application.

We've already discussed why we all should learn to code in Python. This will be the first in a series of exercises to practice coding.

Create a new text file on your desktop named: tobinary.py. Make sure that the file extension is .py and not .txt

Go to the command prompt and type:
cd %homedrive%%homepath%\desktop
dir *.py
notepad tobinary.py

Write the code shown below, take your time. Make sure it's exactly the same. The instructor cannot see your code. Save it. Close Notepad.


first = int(input("Enter the first decimal byte: "))

second = int(input("Enter the second decimal byte: "))

third = int(input("Enter the third decimal byte: "))

fourth = int(input("Enter the fourth decimal byte: "))

print(format(first,'b').zfill(8) + ' ' + format(second,'b').zfill(8) + ' ' + format(third,'b').zfill(8) + ' ' + format(fourth,'b').zfill(8))


On the command line just type:
python tobinary.py


Example: 170.170.170.170 results into 10101010 10101010 10101010 10101010,   try your own numbers.
