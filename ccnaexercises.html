CCNA Labs.
All the exercises assume that we are using Cisco Catalyst 2960 Switches and Cisco 2911 Routers.

Exercise 1:  Assignment of IPv4 addresses to PCs and Cisco routers.
Create the network shown below and assign IPv4 addresses. For each of the routers you can use this syntax:
enable
configure terminal
int gig0/0 ---you'll do the same with the other interfaces gig0/1 and gig0/2
ip address x.y.z.s 255.a.b.c ---where x, y, z, s, a, b, c are shown on the picture.
no shutdown
After configuring each router's interfaces you can save the configuration by typing:
do copy run start ---and press ENTER twice.
Now press CTRL Z
To verify the interfaces' configuration just type:
show ip int brief

Exercise 2:  Testing IPv4 addresses assignment.

On each PC go to the terminal and type: ipconfig
From each PC ping its default gateway (the router IPv4 address): ping w.x.y.z

Exercise 3:  Saving the PKT file and creating copies.

Save the file as myTemplate.pkt. Close the Packet Tracer.
Go to the command line and type:
cd %homedrive%%homepath%\Cisco Packet Tracer 6.2sv\saves
copy myTemplate.pkt staticrouting.pkt
copy myTemplate.pkt riprouting.pkt
copy myTemplate.pkt eigrprouting.pkt
copy myTemplate.pkt ospfrouting.pkt
