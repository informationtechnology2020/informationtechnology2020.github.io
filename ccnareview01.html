CCNA Review.

Assigning passwords to a router.




Download the starting .pkt file from right here.

Step 1:

Go to the laptop and click on the terminal (not on the command prompt!), leave the default settings as they are (bits per second, data bits, parity, stop bits and flow control) and click OK.

Type no when asked Would you like to enter the initial configuration dialog?

Type these commands:

enable

configure terminal

enable secret ccna

line console 0

password ccnp

login

exit

hostname router001

ip domain-name krebner.com

crypto key generate rsa

1024

line vty 0 15

login local

exit

username wendell password odom

int gig0/0

ip address 192.168.1.254 255.255.255.0

no shut

end

copy run start


Step 2:

Now go the command prompt of the PC and type:

ipconfig

ping 192.168.1.254

ssh -l wendell 192.168.1.254

odom

en

ccna

show run | section line con 0

configure terminal

service password-encryption

exit

show run | section line con 0


Step 3:

Close the .pkt file and save it.
