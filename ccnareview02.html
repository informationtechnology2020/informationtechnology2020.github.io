CCNA Review.

IPv4 static and dynamic routing.




Download the starting .pkt file from right here.

Static Routing.

Type on the left router:

enable

configure terminal

ip route 0.0.0.0 0.0.0.0 10.0.0.2

end

copy run start

Type on the middle router:

enable

configure terminal

ip route 192.168.1.0 255.255.255.0 10.0.0.1

ip route 192.168.3.0 255.255.255.0 20.0.0.2

end

copy run start

Type on the right router:

enable

configure terminal

ip route 0.0.0.0 0.0.0.0 20.0.0.1

end

copy run start

Test routing. Just ping the server from the PCs.

RIPv2 Dynamic Routing.

Type on the left router:

enable

configure terminal

no ip route 0.0.0.0 0.0.0.0 10.0.0.2

router rip

version 2

network 192.168.1.0

network 10.0.0.0

end

copy run start

Type on the middle router:

enable

configure terminal

no ip route 192.168.1.0 255.255.255.0 10.0.0.1

no ip route 192.168.3.0 255.255.255.0 20.0.0.2

router rip

version 2

network 192.168.2.0

network 10.0.0.0

network 20.0.0.0

end

copy run start

Type on the right router:

enable

configure terminal

no ip route 0.0.0.0 0.0.0.0 20.0.0.1

router rip

version 2

network 192.168.3.0

network 20.0.0.0

end

copy run start

Test routing. Just ping the server from the PCs.

EIGRP Dynamic Routing.

Type on the left router:

enable

configure terminal

no router rip

router eigrp 10

network 192.168.1.0 255.255.255.0

network 10.0.0.0 255.0..0.0

end

copy run start

Type on the middle router:

enable

configure terminal

no router rip

router eigrp 10

network 192.168.2.0 255.255.255.0

network 10.0.0.0 255.0.0.0

network 20.0.0.0 255.0..0.0

OSPF Dynamic Routing.

Type on the left router:

enable

configure terminal

no router eigrp 10

router ospf 11

network 192.168.1.0 255.255.255.0 area 0

network 10.0.0.0 255.0..0.0 area 0

end

copy run start
