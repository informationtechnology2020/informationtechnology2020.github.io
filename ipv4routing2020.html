IPv4 routing (static and dynamic).

Download the starting .pkt template from right here.



Step 1:   Run ipconfig on each PC to verify the IPv4 addresses.

Step 2:   Type show ip int brief on each router's CLI and verify the IPv4 addresses assignment.

Step 3:   Verify connectivity between each PC and its closest router by typing ping ...

Step 4:   Once the previous verifications have been completed, create four copies of the .pkt:  ipv4static.pkt,  ipv4rip.pkt  ipv4eigrp.pkt,  ipv6ospf.pkt.
