Configuring IPv6 routing.

Download the starting .pkt template from right here.

Step 1.  Go to the command prompt on each PC/Server and find out the IPv6 address by typing ipv6config. Notice that we are still missing the default gateway IPv6 address. We first have to assign addresses to the routers.

Step 2.  Go to the CLI of each router and assign IPv6 addresses:

Left router:

enable

configure terminal

int gig0/0

ipv6 address 2020:acad:baba:1::/64 eui-64

no shutdown

exit

int gig0/1

ipv6 address 2020:acad:baba:4::/64 eui-64

no shutdown

exit

end

show ipv6 int brief

Middle router:

enable

configure terminal

int gig0/0

ipv6 address 2020:acad:baba:2::/64 eui-64

no shutdown

exit

int gig0/1

ipv6 address 2020:acad:baba:4::/64 eui-64

no shutdown

exit

int gig0/2

ipv6 address 2020:acad:baba:5::/64 eui-64

no shutdown

exit

end

show ipv6 int brief

Right router:

enable

configure terminal

int gig0/0

ipv6 address 2020:acad:baba:3::/64 eui-64

no shutdown

exit

int gig0/1

ipv6 address 2020:acad:baba:5::/64 eui-64

no shutdown

exit

end

show ipv6 int brief

Step 3.  Copy and paste on Notepad all router's IPv6 addresses. You'll need them.

Step 4.  Go back to each PC/Server and add the default gateway IPv6 address, which is the gig0/0 interface IPv6 address.

Step 5.  Type ipv6config on each PC/Server and make sure that you can ping its default gateway.

Step 6.  Make four copies of the PKT file: ipv6staticrouting.pkt, ipv6ripv6.pkt, ipv6eigrpv6.pkt and ipv6ospfv6.pkt.

Step 7.  Now open each file and configure IPv6 routing (static, RIPv6, EIGRPv6 and OSPFv6). The required syntax is shown below. Take it easy. Think before you type.
