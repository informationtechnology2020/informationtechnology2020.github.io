Simulating a network.


  1. Download the Cisco Packet Tracer from right here and install it. Just click, click, click, ... and disregard any error messages.

  2. Run the Packet Tracer and create the network seen below. Use the Cisco Catalyst 2960 switch. Assign IP addresses to the PCs.

  3. Test communication between PCs. For example, do this on the first PC: ping 192.168.1.20 --- ping the other PCs too.

  4. Add another switch and two laptops. Assign IP addresses:

  5. Test connectivity between devices. Choose any PC and ping a laptop. Choose a PC and ping all the other devices. For example: ping 192.168.1.50

  6. Add another switch and one server. Assign the server IP address as seen on the picture below:

  7. Test connectivity to the server. Just choose any PC or laptop and ping it: ping 192.168.1.200

  8. Now, we'll test HTTP traffic. Choose any PC or laptop and use the web browser to type: http://192.168.1.200

  9. Finally, we'll test FTP traffic. Choose any PC or laptop and type on the command prompt: ftp 192.168.1.200 . Use cisco as both the username and the password. You won't see the password. Just type it and click ENTER.

  10. Save the file as Exercise1.pkt.
