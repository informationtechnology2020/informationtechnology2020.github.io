Introduction to routing.

.


  1. Download the starting Packet Tracer file from right here and open it.

  2. Go to each computer (and server too) and find out the IP address, the subnet mask and the default gateway.

  3. Ping the default gateway from each PC (and server).

  4. From each PC ping the server.

  5. From each PC access the web server. That is, use the web browser to access the default web site. In few words, go to http://192.168.2.200 from each PC's web browser.

  6. Trace the route taken from each PC to the server. Just use the tracert command.

  7. That's it. Close the file.
